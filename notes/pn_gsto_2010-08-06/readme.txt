This folder contains the following files:


*farquar.f90
 
The F sourcecode for the Farquar model



*a.exe

The exetutable of this sourcecode



*mod_pars.txt

A file containing the values of the species specific m0del parameters seperated 
by spaces in the following order:

alpha g_0 m V_cmax_25 J_max_25



*input.csv

A *.csv file containing a list of input parameters in the order:

c_i T Q

This data is actually the spanish wheat dataset (containing all values).





*output.csv

A *.csv file containing the current model results in the following order:

A_c A_q A_n
